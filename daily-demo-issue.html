Daily Demo Issue Generator

1. General Information

This ScriptRunner scheduled job automatically creates a demo issue in my Automation Showcase (AS) Jira project. It is part of my Jira Automation Portfolio and demonstrates:

  • Scheduled automation in Jira Cloud
  • Using Groovy with ScriptRunner
  • Calling the Jira Cloud REST API (/rest/api/3/issue)
  • Basic logging of success and error cases

2. Execution Flow

  1. The ScriptRunner job is triggered (scheduled daily or manually using Run now).
  2. The script builds a summary that includes the current date and time.
  3. The script creates an Atlassian Document Format (ADF) description with a short explanation and a link to my portfolio site.
  4. The script sends a POST request to /rest/api/3/issue with the JSON payload.
  5. If the request is successful, the script logs the new issue key (for example, AS-23).
  6. If there is an error, the script logs the HTTP status and the error body for troubleshooting.

3. Script Structure

  • Configuration: project key (AS) and issue type (Task).
  • Summary: text including the current timestamp ("Issue created with ScriptRunner - yyyy-MM-dd HH:mm").
  • Description: wrapped in ADF, mentioning that it is part of my Jira Automation Portfolio and signed as Created by Elva Ramos.
  • REST Call: uses post("/rest/api/3/issue") and JsonOutput.toJson(payload) inside ScriptRunner.
  • Logging: logs a success message with the issue key, or a warning with the status and response body.

4. Example Output

When the job runs, a new issue is created in the AS project, with:

  • A summary such as Issue created with ScriptRunner - 2025-12-09 19:30
  • A description explaining that it was generated automatically by a ScriptRunner job as part of my Jira Automation Portfolio
