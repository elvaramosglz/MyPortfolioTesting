Daily Demo Issue Generator

ScriptRunner scheduled job that automatically creates a demo issue in my Automation Showcase (AS) Jira project. This is part of my Jira Automation Portfolio.

1. General Information

This job runs in ScriptRunner for Jira Cloud and creates a new issue in the AS project either on a schedule or via Run now. It showcases:

  • Using Groovy inside ScriptRunner.
  • Calling the Jira Cloud REST API (/rest/api/3/issue).
  • Working with Atlassian Document Format (ADF) for descriptions.
  • Logging success and error cases for troubleshooting.

2. Execution Flow

  1. The ScriptRunner job is triggered (scheduled daily or manually).
  2. The script builds a summary that includes the current date and time.
  3. An ADF description is generated with a short explanation and a link to my portfolio.
  4. The script sends a POST request to /rest/api/3/issue with the JSON payload.
  5. On success, the script logs the new issue key (for example, AS-23).
  6. On failure, it logs the HTTP status code and the response body.

3. Script Structure

  • Configuration: project key (AS) and issue type (Task).
  • Summary: "Issue created with ScriptRunner - yyyy-MM-dd HH:mm".
  • Description: ADF document that includes:
    • A note that it was generated automatically by ScriptRunner.
    • A signature: Created by Elva Ramos.
    • A link to my portfolio website.
  • API Call: post("/rest/api/3/issue") + JsonOutput.toJson(payload) inside ScriptRunner.
  • Logging: success log with the issue key; warning log with status and body on error.

4. Example Output

Each time the job runs, a new issue appears in the AS project with:

  • A summary like Issue created with ScriptRunner - 2025-12-09 19:30.
  • A description explaining that this is an automated demo issue, created by Elva Ramos, with a link to my portfolio site.
